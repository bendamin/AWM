About Page

This is a sample web mapping project by Ben Cox -
A student in DT228 Year 4.

This about page is an example page with information about the site:

  • The site is hosted by a Digital Oceans docker droplet and uses a domain from GoDaddy.
  • The domain is bcox.xyz and utilizes https encryption using nginx and certbot
  • It is deployed as a progressive web app and can be installed on mobile or desktop
  • The map will show your current location and update as you move
  • Clicking anywhere on the map will drop a marker and get you travel information on how long it taks to get their by various methods
  • The site can show and store your recent journeys but requires you to register and login first
  • Travel information is retrieved via the Google Distance Matrix API
  • While logging in please follow the criteria for passwords and usernames


  • The site was built and tested on a local machine and once finished, it was built using bcox.xyz

The backend uses geoDjango and python
